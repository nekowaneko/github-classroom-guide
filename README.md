## 如何上傳檔案至此
1.在 GitHub 網站上直接點擊「Add file」按鈕

2.選擇「Upload files」上傳文件

3.或選擇「Create new file」創建新文件

4.完成後點擊「Commit changes」提交更改

## 如何下載此處檔案
1.在你要下載的檔案上點一下

2.看見右上角有三個點，點一下

3.點選download，完成

## 檔案取名格式(非常重要!!)
如果你是二愛第二組，你們組的簡報(pdf)取名就是"二愛第二組"。
